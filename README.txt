1. Go to catkin_ws/src/turtlebot3_simulations/turtlebot3_gazebo
2. Copy exactly the same folder from github to the folder you have opened latestly.

Option for running world.launch file
1. Don't forget to export TURTLEBOT3_MODEL=waffle_pi to use a camera
2. roslaunch turtlebot3_gazebo <name of world>.launch
